## Bases de Cálculo

### 1. **FGTS** (Fundo de Garantia do Tempo de Serviço)

- **Trabalhadores em geral**: **8%** do salário mensal.
- **Trabalhadores domésticos**: **11,2%** do salário mensal.
- **Jovem aprendiz**: **2%** do salário mensal.
